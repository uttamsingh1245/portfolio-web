# Uttam Singh — 3D Portfolio

A modern, deep-space themed portfolio built with **React + Vite**.

## ✨ Features
- Animated particle field with connection lines
- 3D tilt-on-hover cards (CSS perspective transform)
- Custom cursor with lag-ring effect
- Typewriter role animation in hero
- Scroll-triggered section reveal animations
- Ambient glow orbs & mesh gradients
- Timeline-based education section
- Fully responsive

## 🚀 Getting Started

```bash
# 1. Install dependencies
npm install

# 2. Start dev server
npm run dev

# 3. Build for production
npm run build

# 4. Preview production build
npm run preview
```

Then open **http://localhost:5173** in your browser.

## 📁 File Structure
```
portfolio/
├── index.html
├── vite.config.js
├── package.json
└── src/
    ├── main.jsx      # Entry point
    ├── App.jsx       # Entire portfolio (single file)
    └── index.css     # Global styles & CSS variables
```

## 🎨 Customisation
All content is in the `DATA` object at the top of `src/App.jsx`.  
Update links, projects, skills etc. there.
